# Daometer Client

## Setup

```bash
cd /Users/chigozzdev/Desktop/daometer/client
npm install
cp .env.example .env
npm run dev
```

## Environment

- `VITE_API_BASE_URL` (default `http://localhost:4000/api/v1`)

## Routes

- `/` landing
- `/auth` login/register
- `/dashboard` overview
- `/dashboard/flows`
- `/dashboard/notifications`
- `/dashboard/executions`
- `/dashboard/others`
- `/dashboard/settings`
